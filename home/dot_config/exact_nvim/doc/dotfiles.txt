*dotfiles.txt* Neovim config from dotfiles.

==============================================================================
MAPPINGS                                                  *dotfiles-mappings*

Mappings are nonrecursive unless otherwise noted.
Shifted `<Leader>` mappings also have an unshifted alias using
`<Leader><Leader>` plus the unshifted key when possible.

                                                        *yoP*
yoP                     Toggle system clipboard mappings between the `+`
                        and `*` registers. See |quote| and |clipboard|.

                                                        *yom*
yom                     Toggle `colorcolumn` between `81` and disabled.
                        See |'colorcolumn'|.

                                                        *q*
q                       Move backward to the end of the previous word
                        (recursive). Equivalent to |ge|.

                                                        *Q*
Q                       Move backward to the end of the previous WORD
                        (recursive). Equivalent to |gE|.

                                                        *U*
U                       Redo. Equivalent to |CTRL-R|.

                                                        *&*
&                       Repeat the last substitution with its flags.
                        Equivalent to |:&&|.

                                                        *<F5>*
<F5>                    Restart Neovim and reload the configuration.
                        See |:restart|.

                                                        *<BS>*
<BS>                    In insert mode, move one character left.
                        Equivalent to |i_<Left>|.
                        In command-line mode, move one character left.
                        Equivalent to |c_<Left>|.

                                                        *<CR>*
<CR>                    In insert mode, exit to normal mode.
                        Equivalent to |i_<Esc>|.
                        In normal and visual mode, enter command-line mode.
                        See |:|.
                        In the command-line window and quickfix buffers,
                        keep the default `<CR>` behavior.

                                                        *<C-CR>*
<C-CR>                  Split the line at the cursor from normal mode and
                        enter insert mode (recursive). Uses |i_<CR>|.

                                                        *<S-Esc>*
<S-Esc>                 Toggle the command-line window from normal mode.
                        See |q:| and |cmdwin|.

                                                        *<C-A>*
<C-A>                   In command-line mode, jump to the start of the
                        command line. Equivalent to |c_<Home>|.

                                                        *<C-G>*
<C-G>                   Jump to the middle of the screen. Equivalent to |M|.

                                                        *<C-H>*
<C-H>                   In insert mode, backspace.
                        Equivalent to |i_<BS>|.
                        In command-line mode, backspace.
                        Equivalent to |c_<BS>|.

                                                        *<C-J>*
<C-J>                   In command-line mode, move forward in command
                        history. See |c_CTRL-N| and |cmdline-history|.

                                                        *<C-K>*
<C-K>                   In insert mode, split the line at the cursor.
                        Uses |i_<CR>|.
                        In command-line mode, move backward in command
                        history. See |c_CTRL-P| and |cmdline-history|.

                                                        *<C-L>*
<C-L>                   In insert mode, move one character right.
                        Equivalent to |i_<Right>|.
                        In command-line mode, move one character right.
                        Equivalent to |c_<Right>|.

                                                        *<C-Q>*
<C-Q>                   Replay register `q`. See |@| and |recording|.

                                                        *<C-R>*
<C-R>                   Restore the current line to its state before the
                        last change. See |undo-redo|.

                                                        *<Tab>*
<Tab>                   Start macro recording using register `q`.
                        Equivalent to |recording|.

                                                        *<Tab><Tab>*
<Tab><Tab>              Start recording a macro into register `q`.
                        Uses |recording|.

                                        *<Leader>h* *<Leader>j*
                                        *<Leader>k* *<Leader>l*
<Leader>{hjkl}          Split the current window in the given direction.
                        Uses |:split|, |:vsplit|, |:leftabove|, and
                        |:rightbelow|.

                                        *<Leader>H* *<Leader>J*
                                        *<Leader>K* *<Leader>L*
                                        *<Leader><Leader>h*
                                        *<Leader><Leader>j*
                                        *<Leader><Leader>k*
                                        *<Leader><Leader>l*
<Leader>{HJKL}          Split the frame at the outer edge in the given
                        direction. Uses |:split|, |:vsplit|, |:topleft|,
                        and |:botright|.

                                                *<Leader><Tab>*
<Leader><Tab>           Open a new tab. Equivalent to |:tabnew|.

                                                *<Leader><S-Tab>*
                                                *<Leader><Leader><Tab>*
<Leader><S-Tab>         Close the current tab. Equivalent to |:tabclose|.

                                                        *<C-,>*
<C-,>                   Go to the previous tab. Equivalent to |gT|.

                                                        *<C-.>*
<C-.>                   Go to the next tab. Equivalent to |gt|.

                                        *<Leader>:* *<Leader>/* *<Leader>?*
                                        *<Leader><Leader>/*
<Leader>:               Open the command-line history window.
                        Equivalent to |q:|.
<Leader>/               Open the search history window for `/`.
                        Equivalent to |q/|.
<Leader>?               Open the search history window for `?`.
                        Equivalent to |q?|.

                                                        *<Leader>xz*
<Leader>xz              Close the preview window. Equivalent to |:pclose|.

                                                        *<Leader>o*
<Leader>o               Clear search highlighting. Equivalent to
                        |:nohlsearch|.

                                                        *<Leader>n*
<Leader>n               Open a new empty buffer. Equivalent to |:enew|.

                                                        *<Leader>E*
                                                *<Leader><Leader>e>*
<Leader>E               Reload the current file, discarding changes.
                        Equivalent to |:edit!|.

                                                        *<Leader>s*
<Leader>s               Write the file only if it changed.
                        Equivalent to |:update|.

                                                        *<Leader>S*
                                                *<Leader><Leader>s>*
<Leader>S               Force write the file. Equivalent to |:w!|.

                                                *<Leader><CR>*
<Leader><CR>            Quit the current window. Equivalent to |:quit|.

                                                        *<Leader>q*
<Leader>q               Quit all windows. Equivalent to |:quitall|.

                                                        *<Leader>Q*
                                                *<Leader><Leader>q>*
<Leader>Q               Force quit all windows. Equivalent to |:quita!|.

                                                        *<Leader>=*
<Leader>=               Start `:put =` from the expression register.
                        See |:put| and |quote=|.

                                        *<Leader>c* *<Leader>C*
                                        *<Leader><Leader>c*
                                        *<Leader>cc*
<Leader>c{motion}       Yank to the mapped system clipboard register.
                        See |quote| and |clipboard|.
<Leader>C               Yank to end of line to the mapped system clipboard
                        register. See |quote| and |clipboard|.
<Leader>cc              Yank the current line to the mapped system clipboard
                        register. See |quote| and |clipboard|.

                                        *<Leader>m* *<Leader>M*
                                        *<Leader><Leader>m*
                                        *<Leader>mm*
<Leader>m{motion}       Delete to the mapped system clipboard register.
                        See |quote| and |clipboard|.
<Leader>M               Delete to end of line to the mapped system clipboard
                        register. See |quote| and |clipboard|.
<Leader>mm              Delete the current line to the mapped system clipboard
                        register. See |quote| and |clipboard|.

                                        *<Leader>p* *<Leader>P*
                                        *<Leader><Leader>p*
<Leader>{pP}            Paste from the mapped system clipboard register.
                        See |quote| and |clipboard|.

                                                *<C-;>* *<C-;><C-;>*
<C-;>                   Use the mapped system clipboard register with the
                        substitution operator (recursive). See |quote|.
<C-;><C-;>              Substitute the whole line with the mapped system
                        clipboard register (recursive). See |quote|.
